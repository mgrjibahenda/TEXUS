# Midnight Bluff Tavern

这是一个可直接上传到 GitHub Pages 的 3D 网页酒馆心理牌局原型。

## 使用方法

1. 解压 zip。
2. 把 `index.html`、`style.css`、`game.js` 三个文件拖进你的 GitHub 仓库根目录。
3. Commit changes。
4. 在 GitHub 仓库 Settings → Pages 中启用 Pages。
5. 打开 GitHub Pages 链接即可游玩。

## 注意

- 这是原创网页原型，没有使用任何商业游戏的模型、贴图、音乐或 UI 素材。
- 运行时会从 CDN 加载 Three.js，所以 GitHub Pages 打开时需要联网。
- 如果你想完全离线运行，可以之后把 Three.js 下载到本地并修改 `game.js` 的 import 地址。
