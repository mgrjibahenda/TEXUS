# Midnight Bluff Tavern

这是一个可直接上传到 GitHub Pages 的自包含 3D 网页牌局原型。

## 使用方法
1. 解压 zip。
2. 把 index.html、style.css、game.js、README.md 全部拖进 GitHub 仓库根目录。
3. Commit changes。
4. 在 Settings -> Pages 打开 GitHub Pages。

## 注意
这个版本不依赖 Three.js CDN，也不需要安装任何包。画面使用 CSS 3D 和原生 JavaScript 渲染，所以更容易在 GitHub Pages 直接显示。
