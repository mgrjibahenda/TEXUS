# Texas Hold'em v4 Premium

Render 可部署升级版。

新增：
- 进入界面只需要 Player name。
- 删除 Custom ID 和多余按钮。
- Dealer 可以开局前设置每个玩家筹码。
- Dealer 可以一键把一个数字设置给所有玩家。
- 玩家均匀分布在圆形牌桌周围。
- 非本人回合不显示行动按钮。
- 本人回合只显示当前可用操作。
- 10 不再显示为 T。
- 不同牌型触发不同特效，高级牌型更华丽。
- 赢家吹捧和破产嘲讽随机变化。
- 一局结束后只有 Dealer 有 Next Hand。

Render 设置：
Build Command: npm install
Start Command: npm start
